# Legir Icon Directions

这 3 个草案都围绕同一组约束：

- `reader-first`
- `lightweight`
- `precise`
- `AI` 只作为轻微气质，不作为主图形

文件：

- `legir-icon-concept-a.svg`
- `legir-icon-concept-b.svg`
- `legir-icon-concept-c.svg`

## A

`Page Fold + L`

- 最稳
- 最像应用图标
- 兼顾 `Legir` 的品牌字母和 PDF 页面语义
- 小尺寸识别最好

这是我当前最推荐的一版。

## B

`Bookmark + L`

- 更像阅读器 / 书签工具
- 品牌感更柔和
- 比 A 更偏产品化，但和 PDF 的直接联系稍弱

## C

`Margin Rail + Reading Window`

- 最强调“阅读焦点”
- 有一点“导览 / 批注 / AI 阅读窗口”的感觉
- 视觉比较现代，但比 A 更依赖细节

## 下一步

如果你要继续，我建议直接在这 3 个里选一条路线：

1. 保守、稳、适合正式上线：`A`
2. 更像阅读产品：`B`
3. 更强调智能阅读工作流：`C`

选定后我可以继续做：

- 单色版
- macOS / Windows 风格适配版
- favicon 简化版
- Tauri 正式图标替换稿
